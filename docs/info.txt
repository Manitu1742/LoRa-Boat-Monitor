Merge all binarys for a merged-firmware.bin in Gitpod
#####################################################

* Go to /workspace/LoRA-Boat-Monitor/docs and run following commands

pip install esptool
esptool.py --chip esp32 merge_bin -o merged-firmware.bin --flash_mode dio --flash_freq 40m --flash_size 4MB 0x1000 bootloader.bin 0x8000 partitions.bin 0x10000 firmware.bin
